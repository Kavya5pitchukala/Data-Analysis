# 📊 Data Analysis & Visualization Dashboard

An interactive data analysis dashboard that processes large datasets using SQL queries and Excel-style logic, surfacing actionable business insights through dynamic charts and structured visual reports.

## 🚀 Features

- Interactive charts: bar, line, pie, and summary tables
- SQL-style data aggregation (GROUP BY, ORDER BY, filters)
- KPI summary cards with trend indicators
- Export-ready data views
- Responsive layout for all screen sizes

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Data Processing:** SQL query simulation in JS
- **Concepts:** Data Analysis, Aggregation, Visualization, Business Intelligence

## 📁 Project Structure

```
data-analysis-dashboard/
├── index.html          # Main dashboard
├── css/
│   └── style.css
├── js/
│   ├── data.js         # Dataset + SQL-style aggregation functions
│   ├── charts.js       # Chart rendering (Canvas API)
│   └── dashboard.js    # UI controller
└── README.md
```

## ⚙️ Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/data-analysis-dashboard.git
   cd data-analysis-dashboard
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```
